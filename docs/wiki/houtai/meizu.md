# 📱魅族手机设置教程
> 测试机型：魅族16s Pro<br/>
系统版本：Android 9<br/>
## 💡设置步骤

打开手机管家，点击权限管理

<center>

![叮小跳](https://b.dinglegedong.com/img/meizu/1.jpg)<br/>

</center>

<center>

![叮小跳](https://b.dinglegedong.com/img/meizu/2.jpg)<br/>

</center>

找到“叮小跳”，点击打开

<center>

![叮小跳](https://b.dinglegedong.com/img/meizu/3.jpg)<br/>

</center>

选择允许后台运行

<center>

![叮小跳](https://b.dinglegedong.com/img/meizu/4.jpg)<br/>

</center>
